# PLP Peer Exercise

Peer Exercise in COBOL for seminar: Programming Languages and Paradigms

* `report_generator.cbl`: the COBOL program
* `report.dat`: the generated result

To compile and run the code, please execute the following commands (take macOS platform for example, please refer to the tutorial for setting up COBOL on other platforms):

```bash
brew install gnucobol
cobc -x report_generator.cbl
./report_generator
```
